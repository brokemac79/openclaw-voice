# Just Use OpenClaw Voice

Pick one path and open only that guide.

1. **Use it in browser** - If someone else already runs OpenClaw Voice for you, start here for quick setup and troubleshooting: `docs/user-guide.md`.
2. **Host it yourself** - If you want to run OpenClaw Voice on this computer, use the copy-and-paste setup steps: `docs/host-it-yourself.md`.
3. **Deploy for other devices** - If your local setup already works and now you want phones or other devices to reach it, follow this deployment guide: `docs/deployment-guide.md`.
